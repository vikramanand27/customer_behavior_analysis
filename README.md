# 📊 Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow, covering data extraction, cleaning, exploration, analysis, visualization, and reporting. The project leverages Python for data processing, PostgreSQL for querying and analysis, Power BI for interactive dashboards, and Gamma for business presentations.

The objective was to transform raw data into actionable insights that support data-driven decision-making.

---

## Dataset

**Source:** Publicly available dataset (online source)

**Description:**
The dataset contains business-related information used for performing exploratory data analysis (EDA), SQL-based analysis, and dashboard creation.

**Key Attributes:**

* Customer Information
* Product Details
* Sales Data
* Transaction Records
* Performance Metrics

## Project Workflow

### 1. Data Loading

* Imported dataset using Python.
* Inspected data structure and column information.
* Verified data quality and completeness.

### 2. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent formats.
* Standardized column names and data types.

### 3. Exploratory Data Analysis (EDA)

* Generated descriptive statistics.
* Analyzed trends, patterns, and distributions.
* Identified key business metrics.
* Created visualizations to uncover insights.

### 4. SQL Analysis (PostgreSQL)

* Loaded cleaned data into PostgreSQL.
* Wrote SQL queries to:

  * Analyze customer behavior
  * Identify top-performing products
  * Measure sales performance
  * Calculate business KPIs
  * Generate summary reports

### 5. Dashboard Development (Power BI)

Created an interactive dashboard featuring:

* KPI Cards
* Sales Analysis
* Product Performance
* Customer Insights
* Trend Analysis
* Filters and Slicers for detailed exploration

### 6. Reporting

* Documented findings and recommendations.
* Summarized key business insights.
* Created stakeholder-friendly reports.

### 7. Presentation (Gamma)

* Developed a professional presentation.
* Highlighted project objectives, methodology, insights, and recommendations.
* Presented findings in a visually engaging format.

## How to Run

### Prerequisites

* Python 3.x
* PostgreSQL
* Power BI Desktop
* Jupyter Notebook

### Steps

1. Clone the repository:

```bash
git clone https://github.com/your-username/data-analytics-project.git
```

2. Install required Python libraries:

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2
```

3. Open Jupyter Notebook and run:

```bash
jupyter notebook
```

4. Execute:

   * Data Loading Notebook
   * Data Cleaning Notebook
   * EDA Notebook

5. Import cleaned data into PostgreSQL.

6. Run SQL queries from:

```sql
SQL/Analysis_Queries.sql
```

7. Open the Power BI dashboard:

```text
PowerBI/Dashboard.pbix
```

8. Review the final report and presentation.
